Adicionar uma nova linha

Adicionar uma nova linha é tão simples quanto inserir um novo objeto no dataSourcearray.

Uma rápida recapitulação do tutorial anterior dataSourceé a lista de dados primária usada no elemento de tabela. Nós o atribuímos diretamente ao USER_DATAarray, que são apenas dados fictícios estáticos. Isso virá do back-end por meio de uma chamada de API em um cenário da vida real.

Vamos aos passos necessários:

  1. Adicione um novo buttoncom um clickevento logo antes do tablecomponente. Vamos envolvê-lo em um articlecontêiner para poder adicionar mais botões de ação em etapas posteriores:
<article class="table-header">
  <button class="button-add-row" mat-button (click)="addRow()">Add Row</button>
</article>
<table mat-table [dataSource]="dataSource">
  <ng-container [matColumnDef]="col.key" *ngFor="let col of columnsSchema">
    <th mat-header-cell *matHeaderCellDef>
      {{ col.label }}
    </th>
    <td mat-cell *matCellDef="let element">
      <div [ngSwitch]="col.type" *ngIf="!element.isEdit">
        <div class="btn-edit" *ngSwitchCase="'isEdit'">
          <button mat-button (click)="element.isEdit = !element.isEdit">
            Edit
          </button>
        </div>
        <span *ngSwitchDefault>
          {{ element[col.key] }}
        </span>
      </div>
      <div [ngSwitch]="col.type" *ngIf="element.isEdit">
        <div class="btn-edit" *ngSwitchCase="'isEdit'">

        </div>
        <mat-form-field *ngSwitchCase="'date'" appearance="fill">

  1. Implemente addRowa função para inserir um novo objeto na dataSourcematriz com valores vazios. Certifique-se de que contém a isEditpropriedade definida como true para torná-la editável para o usuário:
import { Component } from '@angular/core';

const USER_DATA = [
  {"name": "John Smith", "occupation": "Advisor", "dateOfBirth": "1984-05-05", "age": 36},
  {"name": "Muhi Masri", "occupation": "Developer", "dateOfBirth": "1992-02-02", "age": 28},
  {"name": "Peter Adams", "occupation": "HR", "dateOfBirth": "2000-01-01", "age": 20},
  {"name": "Lora Bay", "occupation": "Marketing", "dateOfBirth": "1977-03-03", "age": 43},
]

const COLUMNS_SCHEMA = [
  {
      key: "name",
      type: "text",
      label: "Full Name"
  },
  {
      key: "occupation",
      type: "text",
      label: "Occupation"
  },
  {
    key: "dateOfBirth",
    type: "date",
    label: "Date of Birth"
},
  {
      key: "age",
      type: "number",
      label: "Age"
  },
  {
    key: "isEdit",
    type: "isEdit",
    label: ""
  }
]

@Component({
  selector: 'app-root',
  templateUrl: './app.component.html',
  styleUrls: ['./app.component.scss']
})
export class AppComponent {
  displayedColumns: string[] = COLUMNS_SCHEMA.map(col => col.key);
  dataSource = USER_DATA;
  columnsSchema: any = COLUMNS_SCHEMA;

  addRow() {
    const newRow = {"name": "", "occupation": "", "dateOfBirth": "", "age": 0, isEdit: true}
    this.dataSource = [...this.dataSource, newRow];
  }
}

Para o Angular acionar a OnPushdetecção de alteração, precisamos clonar o array. É por isso que estamos usando o operador spread no arquivo dataSource.

Em alguns casos, a tabela pode ser longa com muitas linhas e adicionar uma nova entrada na parte inferior exigirá que o usuário role para baixo todas as vezes. Podemos resolver esse problema simplesmente trocando as entradas no novo array que acabamos de criar:

this.dataSource = [newRow, ...this.dataSource];

Remover várias linhas

Esta seção apresentará um novo recurso para oferecer suporte à remoção de várias linhas. Isso requer uma pequena alteração no modelo para acompanhar as linhas selecionadas. Vamos aos passos necessários:

  1. Adicione uma nova isSelectedcoluna ao arquivo COLUMNS_SCHEMA. Isso nos ajudará a determinar quais linhas são selecionadas:

    key: 'isSelected',
    type: 'isSelected',
    label: '',
  },
  {
    key: 'name',
    type: 'text',
    label: 'Full Name',
  },
  {
    key: 'occupation',
    type: 'text',
    label: 'Occupation',
  },
  {
    key: 'dateOfBirth',
    type: 'date',
    label: 'Date of Birth',
  },
  {
    key: 'age',
    type: 'number',
    label: 'Age',
  },
  {
    key: 'isEdit',
    type: 'isEdit',
    label: '',
  }
];

  1. Adicione um elemento de caixa de seleção como uma nova coluna somente dentro da seção do modo sem edição porque não precisamos que apareça durante a edição da linha

  2. Defina um changeevento para inserir a isSelectedpropriedade na linha dinamicamente. $event.checkeddetermina o valor (verdadeiro ou falso)

<div [ngSwitch]="col.type" *ngIf="!element.isEdit">
  <ng-container *ngSwitchCase="'isSelected'">
    <mat-checkbox (change)="element.isSelected = $event.checked"></mat-checkbox>
  </ng-container>
  <div class="btn-edit" *ngSwitchCase="'isEdit'">
    <button mat-button (click)="element.isEdit = !element.isEdit">Edit</button>
    <button mat-button class="button-remove" (click)="removeRow(element.id)">
      Delete
    </button>
  </div>
  <span *ngSwitchDefault>
    {{ element[col.key] }}
  </span>
</div>
  1. Adicione um novo botão remover com um clickevento. Vamos colocá-lo ao lado do botão adicionar criado anteriormente:
<article class="table-header">
  <button class="button-remove-rows" mat-button (click)="removeSelectedRows()">
    Remove Rows
  </button>
  <button class="button-add-row" mat-button (click)="addRow()">Add Row</button>
</article>
  1. Crie uma nova removeSelectedRowsfunção que filtre e retorne as linhas não selecionadas do sourceDataarray:
removeSelectedRows() {
  this.dataSource = this.dataSource.filter((u: any) => !u.isSelected);
}

Vamos esclarecer um pouco mais; quando o usuário seleciona uma linha, o objeto será anexado isSelected: trueao objeto da seguinte forma:

{id: 1, "name": "John Smith", "occupation": "Advisor", "dateOfBirth": "1984-05-05", "age": 36, isSelected: true}

Em seguida, filtramos os valores dataSourceby isSelectedpara removê-los e retornamos uma nova matriz.

Diálogo de confirmação

Se a função remover linhas acionar uma chamada de API para excluir as linhas selecionadas, é bom adicionar uma caixa de diálogo de confirmação antes de continuar. Isso requer alguns passos simples:

  1. Crie um novo ConfirmDialogcomponente. Podemos usar o comando CLI:
ng generate component confirm-dialog
  1. Adicione o modelo necessário para a caixa de diálogo de confirmação. No nosso caso, é apenas uma caixa de diálogo simples de confirmar/cancelar:
<h1 mat-dialog-title>Are you sure you want to delete?</h1>
<div mat-dialog-actions>
    <button mat-button [mat-dialog-close]>Cancel</button>
    <button class="delete-button" mat-button [mat-dialog-close]="true">Delete</button>
</div>
  1. Em app.component.ts, inscreva-se no afterClosedevento e remova todas as linhas selecionadas se a ação for “Sim”, caso contrário, basta fechar a caixa de diálogo:
 removeSelectedRows() {
    this.dialog
      .open(ConfirmDialogComponent)
      .afterClosed()
      .subscribe((confirm) => {
        if (confirm) {
          this.dataSource = this.dataSource.filter((u: any) => !u.isSelected);
        }
      });
  }

Não precisamos adicionar nenhum código ao confirm-dialog.component.tsarquivo porque os botões cancelar/excluir no modelo de diálogo de confirmação contêm uma mat-dialog-closediretiva que fecha automaticamente o diálogo para nós, e isso é tudo de que precisamos por enquanto.
